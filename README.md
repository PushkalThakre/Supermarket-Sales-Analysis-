# Supermarket-Sales-Analysis-
This project analyzes supermarket sales data to identify sales trends, top-performing product categories, and branch performance using Python data analysis and visualization techniques.

Overview-
This project analyzes supermarket sales data to understand sales trends, product performance, and branch revenue. The goal is to extract useful business insights using data analysis techniques.

Tools Used-
Python
Pandas
Matplotlib
Seaborn
Jupyter Notebook 

Analysis Performed-
Sales trend analysis
Top-selling product categories
Sales distribution by category
Branch performance comparison

Conclusion-
The analysis helps identify high-performing products and branches, supporting better business decisions and improved sales strategies.
